# SAMAKY Features

## 🏠 Home Tab

### Foundation Dashboard
- **Total Giving Display**: Large, prominent display of total contributions
- **Foundation Card**: Personalized foundation with your name (e.g., "The Chen Foundation")
- **Quick Stats**: Number of causes, nonprofits, and founding year
- **Gradient Design**: Beautiful forest green gradient with floral watermark

### Match Day Alert
- **Real-time Alerts**: Prominent purple banner for match opportunities
- **GiveMN Integration**: Notifications for Minnesota match days
- **Countdown Timers**: "Give before midnight!" urgency messaging

### Nearby Opportunities ⭐ NEW
- **Volunteer Listings**: In-person volunteer opportunities from verified partners
- **Partner Integration**: 
  - Hands On Twin Cities
  - Catchafire
  - Compass Pro Bono
- **Opportunity Cards**: Beautiful gradient cards with:
  - Type badge (Volunteer/Fundraiser)
  - Distance indicator
  - Date/time details
  - Spots available
  - Partner attribution with shield icon
  - Direct sign-up links
- **Horizontal Scroll**: Swipeable carousel of opportunities

### News Feed
- **Curated Updates**: News from supported causes and organizations
- **Source Attribution**: Shows original source (The Guardian, NPR, etc.)
- **Read Time**: Estimated reading time for each article
- **Cause Tagging**: Environment, Animals, Education, Health categories

## 📅 Calendar Tab

### Interactive Calendar
- **Month View**: Clean, minimal calendar grid
- **Event Indicators**: Color-coded dots for events
- **Year Selector**: Toggle between 2025 and 2026
- **Month Navigation**: Previous/Next buttons

### Event Types
- **Match Days**: Purple highlighting with 2x match badges
- **Giving Days**: Green highlighting for special giving campaigns
- **Tax Deadlines**: Important dates for charitable contributions

### Upcoming Events
- **Event List**: Chronological list of upcoming opportunities
- **Event Cards**: 
  - Organization name
  - Event date
  - Event type icon
  - Match multiplier badges

## 🔍 Search Tab

### Interactive US Map
- **Vector Map**: Hand-drawn style US state boundaries
- **State Labels**: All 50 states labeled
- **Pan & Zoom**: Pinch to zoom, drag to pan
- **User Location**: Blue dot showing your position

### Nonprofit Discovery
- **Map Pins**: Color-coded pins for each cause category
  - 🌿 Environment (Green)
  - 🐾 Animals (Orange)
  - 📚 Education (Blue)
  - ❤️ Health (Red)
- **Pin Interaction**: Tap to select, enlarges and shows details
- **Popup Cards**: Bottom sheet with:
  - Organization name
  - Verification badge
  - Location details
  - Cause category
  - EIN number
  - Action buttons (Donate, Volunteer, Add)

### Filtering
- **Cause Filter**: Filter by Environment, Animals, Education, Health, or All
- **Live Location**: Button to center map on user location
- **Results Count**: Shows number of matching nonprofits

### Nonprofit List
- **Scrollable List**: Up to 5 nonprofits shown below map
- **Organization Cards**:
  - Cause icon (SVG, not emoji!)
  - Name with verification badge
  - Location
  - Cause category badge
  - EIN number

## 📬 Inbox Tab

### Dual View
- **Updates Tab**: News and messages from organizations
- **Receipts Tab**: Transaction history and confirmations

### Updates Section
- **Organization Messages**: Updates from supported nonprofits
- **Unread Indicators**: Green dot for new messages
- **Cause Icons**: Visual identification of message source
- **Timestamps**: Relative time (1h ago, 2d ago)

### Receipts Section
- **Tax Deduction Notice**: "All donations are tax-deductible via Daffy DAF"
- **Transaction Cards**:
  - Organization name
  - Donation amount
  - Date
  - Transaction ID
  - Confirmation status
- **Daffy Integration**: Clear attribution to DAF partner

## 👤 Profile Tab

### User Profile
- **Profile Photo**: Large circular avatar
- **Username**: Handle (e.g., @alexchen)
- **Social Stats**: Follower count
- **Member Since**: Account creation date
- **QR Code**: Shareable QR code for profile

### Foundation Overview
- **Foundation Card**: Same gradient design as Home
- **Donut Chart**: Visual breakdown of cause allocation
- **Cause Breakdown**: 
  - Color-coded squares
  - Cause names
  - Percentage allocations
- **Share Button**: "Share my impact" with social sharing

### Statistics Grid
- **Total Given**: Lifetime contributions
- **Nonprofits**: Number of organizations supported
- **Founded**: Year foundation was created

### Auto-Giving Settings
- **Toggle Switch**: Enable/disable auto-giving
- **Amount Display**: Monthly contribution amount
- **Distribution**: Shows how funds are split across causes
- **Next Donation**: Countdown to next automatic contribution
- **Recycle Icon**: Visual indicator of recurring nature

### My Nonprofits
- **Interactive List**: Tap to view all 6 supported organizations
- **Modal View**: Full-screen overlay with:
  - Organization cards
  - Verified badges
  - Location information
  - Cause categories
  - Allocation percentages

## 🎨 Design System

### Colors
- **SAMAKY Green**: #0e7f50 (Primary brand)
- **Forest Green**: #0a5c38 (Darker accent)
- **Green Light**: #e8f5ee (Backgrounds)
- **Beige/Cream**: #ffffff (Main backgrounds)
- **Purple**: #7c3aed (Match days)
- **Orange**: #f97316 (Accents, stars)

### Typography
- **Headers**: 800 weight, 16-22px
- **Body**: 400-600 weight, 12-14px
- **Small Text**: 10-11px for metadata
- **Monospace**: Transaction IDs

### Icons
- **All SVG**: No emojis, professional vector icons
- **Consistent Stroke**: 2px stroke width
- **Cause Icons**: Leaf, Paw, Book, Heart
- **UI Icons**: Home, Calendar, Search, Inbox, Profile

### Components
- **Cards**: 14-20px border radius, subtle shadows
- **Buttons**: 10-12px border radius, bold text
- **Badges**: Small rounded rectangles
- **Verified Badge**: Green checkmark in circle
