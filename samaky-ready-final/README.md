# SAMAKY - Your Personal Foundation

SAMAKY democratizes foundations by enabling young professionals to create personal impact portfolios similar to family foundations. Think "Robinhood for charitable giving" - a freemium SaaS platform that makes giving aspirational and shareable.

## 🌟 Features

### Core Functionality
- **Personal Foundation Dashboard** - View your total giving, causes, and nonprofits at a glance
- **Interactive Map Discovery** - Find verified nonprofits near you with real-time filtering
- **Giving Calendar** - Track important giving days, match opportunities, and deadlines
- **Nearby Opportunities** - Discover verified volunteer opportunities from partners like:
  - Hands On Twin Cities
  - Catchafire
  - Compass Pro Bono
- **Impact Sharing** - Share your foundation's impact on social media
- **Inbox & Updates** - Receive news and updates from supported nonprofits

### Technical Features
- Mobile-first responsive design
- Real-time interactive US map with nonprofit locations
- Donut chart visualizations for cause allocation
- QR code profile sharing
- Auto-giving automation

## 🚀 Quick Start

### Prerequisites
- Node.js 16+ and npm

### Installation

```bash
# Clone the repository
git clone https://github.com/yourusername/samaky-app.git
cd samaky-app

# Install dependencies
npm install

# Start development server
npm run dev
```

Visit `http://localhost:5173` to see the app running.

### Build for Production

```bash
# Create optimized production build
npm run build

# Preview production build locally
npm run preview
```

## 🎨 Design Philosophy

SAMAKY combines clean institutional design (inspired by cultural institutions like the Minneapolis Institute of Art) with modern mobile UX patterns. Key design elements:

- **SAMAKY Green** (#0e7f50) - Primary brand color
- **Cream/Beige** backgrounds for warmth
- **Forest green gradients** for the foundation card
- **Clean, minimal iconography** - No emojis, professional SVG icons throughout

## 🤝 Partner Integration

SAMAKY partners with established volunteer platforms to surface verified opportunities:

- **Hands On Twin Cities** - Community volunteer opportunities
- **Catchafire** - Skills-based volunteering and fundraising
- **Compass Pro Bono** - Professional skills for nonprofits

All opportunities link directly to partner sites for seamless sign-up.

## 💳 Business Model

**Freemium SaaS:**
- **SAMAKY FREE** - Basic giving tracking
- **SAMAKY PLUS** ($6/month) - Enhanced features, auto-giving, impact analytics
- **SAMAKY ENTERPRISE** - Custom solutions for corporate giving programs

**Infrastructure Partner:** Daffy (Donor-Advised Fund) handles the actual fund management while SAMAKY focuses on UX and social features.
